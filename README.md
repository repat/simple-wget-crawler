Simple wget crawler
======
**Simple wget crawler** is a script for downloading files from a webserver. The list of files is given in a file as a commandline parameter like this:

```
$ cat files.txt
picture1.jpg
picture2.jpg
picture3.jpg

$ ./simple-wget-crawler.sh files.txt
```
